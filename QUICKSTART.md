# Quickstart Guide

Get Codebreaker97 running on your local machine in minutes.

## Prerequisites
1. **Python 3.10+** installed and added to PATH.
2. **Ollama** installed with the `llama3` model downloaded (`ollama run llama3`).
3. **Stable Diffusion WebUI (Automatic1111)** installed locally (Optional, but required for the Image Engine).

1. Clone or download this repository to your local machine.
